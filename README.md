## Overview
Created and managed AWS EC2 infrastructure using AWS CloudFormation.

## Technologies Used
- AWS CloudFormation
- EC2
- Security Groups
- VPC
- YAML

## Features
- Infrastructure as Code (IaC)
- Automated EC2 provisioning
- Reproducible infrastructure deployment

## Deployment
1. Navigate to CloudFormation console.
2. Create a new stack.
3. Upload the YAML template.
4. Review and deploy.

## Learning Outcomes
- Hands-on experience with CloudFormation templates
- Stack creation and management
- Infrastructure automation using IaC principles
